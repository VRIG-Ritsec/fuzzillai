(library fundamentals TS v2) Generates a random integer in the closed interval [a, b] . Contents 1 Parameters 2 Return value 3 Remarks 4 Example 5 See also [ edit ] Parameters a, b - integer values specifying the range [ edit ] Return value A random integer i in the closed interval [a, b] , produced using a thread-local instance of std:: uniform_int_distribution < IntType > invoked with the per-thread random number engine . [ edit ] Remarks If IntType is not one of short , int , long , long long , unsigned short , unsigned int , unsigned long , or unsigned long long , the program is ill-formed. The behavior is undefined if a > b .
