header <cstdio> int putchar ( int ch ) ; Writes a character ch to stdout . Internally, the character is converted to unsigned char just before being written. Equivalent to std:: putc ( ch, stdout ) . Contents 1 Parameters 2 Return value 3 Example 4 See also [ edit ] Parameters ch - character to be written [ edit ] Return value On success, returns the written character. On failure, returns EOF and sets the “error” indicator (see std::ferror() ) on stdout .
