header <cstdio> void rewind ( std:: FILE * stream ) ; Moves the file position indicator to the beginning of the given file stream. The function is equivalent to std:: fseek ( stream, 0 , SEEK_SET ) ; , except that end-of-file and error indicators are cleared. The function drops any effects from previous calls to ungetc .
