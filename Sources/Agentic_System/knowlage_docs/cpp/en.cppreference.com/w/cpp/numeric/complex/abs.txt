<std::complex> (C++26) tuple_element <std::complex> (C++26) [edit] Defined in header <complex> template < class T > T abs ( const complex < T > & z ) ; Returns the magnitude of the complex number z . Contents 1 Parameters 2 Return value 3 Example 4 See also [ edit ] Parameters z - complex value [ edit ] Return value If no errors occur, returns the absolute value (also known as norm, modulus, or magnitude) of z . Errors and special cases are handled as if the function is implemented as std:: hypot ( std:: real ( z ) , std:: imag ( z ) ) .
