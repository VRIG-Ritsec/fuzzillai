<std::complex> (C++26) tuple_element <std::complex> (C++26) [edit] Defined in header <complex> template < class T > complex < T > cos ( const complex < T > & z ) ; Computes complex cosine of a complex value z . Contents 1 Parameters 2 Return value 3 Notes 4 Example 5 See also [ edit ] Parameters z - complex value [ edit ] Return value If no errors occur, the complex cosine of z is returned. Errors and special cases are handled as if the operation is implemented by std::cosh (i * z) , where i is the imaginary unit. [ edit ] Notes The cosine is an entire function on the complex plane, and has no branch cuts. Mathematical definition of the cosine is cos z = e iz +e -iz 2 .
