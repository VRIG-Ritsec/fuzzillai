They only need to be sufficient to ensure that for their argument num : If num has a standard (until C++23) floating-point type T , then std :: real ( num ) has the same effect as std :: real ( std:: complex < T > ( num ) ) . Otherwise, if num has an integer type, then std :: real ( num ) has the same effect as std :: real ( std:: complex < double > ( num ) ) .
