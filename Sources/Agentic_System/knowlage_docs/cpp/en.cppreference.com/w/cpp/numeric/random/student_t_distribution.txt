[edit] Defined in header <random> template < class RealType = double > class student_t_distribution ; (since C++11) Produces random floating-point values x , distributed according to probability density function: \(p(x|n) = \frac{1}{\sqrt{n\pi} } \cdot \frac{\Gamma(\frac{n+1}{2})}{\Gamma(\frac{n}{2})} \cdot (1+\frac{x^2}{n})^{-\frac{n+1}{2} } \) p(x|n) = 1 √ nπ · Γ( n+1 2 ) Γ( n 2 ) · ⎛ ⎜ ⎝ 1+ x 2 n ⎞ ⎟ ⎠ - n+1 2 where n is known as the number of degrees of freedom . This distribution is used when estimating the mean of an unknown normally distributed value given n + 1 independent measurements, each with additive errors of unknown standard deviation, as in physical measurements. Or, alternatively, when estimating the unknown mean of a normal distribution with unknown standard deviation, given n + 1 samples. std::student_t_distribution satisfies all requirements of RandomNumberDistribution .
