Checks if the given wide character corresponds (if narrowed) to a hexadecimal numeric character, i.e. one of 0123456789abcdefABCDEF . If the value of ch is neither representable as a wchar_t nor equal to the value of the macro WEOF , the behavior is undefined. [ edit ] Notes std::iswdigit and std::iswxdigit are the only standard wide character classification functions that are not affected by the currently installed C locale.
