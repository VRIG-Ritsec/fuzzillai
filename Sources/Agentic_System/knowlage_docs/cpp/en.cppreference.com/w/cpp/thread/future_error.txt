Defined in header <future> class future_error ; (since C++11) The class std::future_error defines an exception object that is thrown on failure by the functions in the thread library that deal with asynchronous execution and shared states ( std::future , std::promise , etc). Similar to std::system_error , this exception carries an error code compatible with std::error_code .
