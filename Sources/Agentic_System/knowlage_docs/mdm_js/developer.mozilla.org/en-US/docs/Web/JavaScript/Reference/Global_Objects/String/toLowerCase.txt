It’s been available across browsers since ⁨July 2015⁩. Learn more See full compatibility Report feedback The toLowerCase() method of String values returns this string converted to lower case. In this article Try it Syntax Description Examples Specifications Browser compatibility See also Try it const sentence = "The quick brown fox jumps over the lazy dog."; console.log(sentence.toLowerCase()); // Expected output: "the quick brown fox jumps over the lazy dog." Syntax js toLowerCase() Parameters None. Return value A new string representing the calling string converted to lower case. Description The toLowerCase() method returns the value of the string converted to lower case. toLowerCase() does not affect the value of the string str itself. Examples Using toLowerCase() js console.log("ALPHABET".toLowerCase()); // 'alphabet' Specifications Specification ECMAScript® 2026 Language Specification # sec-string.prototype.tolowercase Browser compatibility Loading… See also String.prototype.toLocaleLowerCase() String.prototype.toLocaleUpperCase() String.prototype.toUpperCase() Help improve MDN Was this page helpful to you?
